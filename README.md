# CISC101-Group--PaperSummarizer

This repository contains the complete system prompt and module architecture for a modular research-paper summarizer based on the PS2 specification.

## Files Included

### /system_prompt.md
Contains the final System Prompt generated by the LLM. It includes greeting rules, safety rules, PS2 constraints, output formatting, and all five required modules.

### /modules/

- module1_intake.md — with a quick description of its purpose 
- module2_section.md — with a quick description of its purpose
- module3_guardrails.md — with a quick description of its purpose
- module4_citation.md — LLM generated student designed module
- module5_contributions.md — LLM generated student designed module


## Purpose
This repo demonstrates modular prompt engineering.
